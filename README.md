## How to start ?

- clone the project with : ```git clone [SSH link]```
- go to the project's directory : ```cd ITube```

## How to start ?

- clone the project with : ```git clone [SSH link]```
- go to the project's directory : ```cd ITube```

In the project directory, you can run:

- install dependencies : ```npm install```
- run server : ```npm start```
